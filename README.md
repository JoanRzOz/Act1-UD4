# NotasFinales
## Descripción
Este programa en Java trabaja con un array de notas numéricas y realiza varias operaciones básicas para analizar los resultados de un alumno.

## Cómo ejecutar el programa
### Ejecución desde el terminal
1. Abre la terminal y navega hasta la carpeta donde está el archivo: `cd ruta/del/proyecto/src`
2. Compila el programa: `javac UD4/NotasFinales.java`
3. Ejecuta el programa: `java UD4.NotasFinales`

### Ejecución desde el IDE
1. Abre el IDE.
2. Ubica la clase NotasFinales.
3. Abre el archivo haciendo doble clic o clic derecho.
4. Presiona el botón de ejecutar o el atajo para lo mismo.
